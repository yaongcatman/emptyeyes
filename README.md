### 1. Python 기초 
* **변수(Variable)**: 내가 나중에 다시 쓰고 싶은 소중한 정보를 담아두는 '이름표 붙은 상자'입니다.
* **자료형(Data Types)**: 상자에 무엇을 넣을지 결정하는 규칙입니다.
    * `int`: 숫자 구슬 (1, 2, 3...)
    * `str`: 글자 카드 ("안녕", "원숭이")
    * `bool`: 예/아니오 스티커 (참/거짓)
    * `list`: 여러 물건을 한 줄로 세운 '기차'
    * `dict`: 단어와 뜻이 짝궁인 '낱말 사전'
* **조건문(if/elif/else)**: 갈림길에서 어디로 갈지 정하는 '표지판'입니다. * **반복문(Loops)**: 
    * `for`: "10번만 돌아!"라고 정해진 만큼만 타는 '회전목마'입니다.
    * `while`: "전기가 켜져 있는 동안 계속 돌아!"라고 하는 '선풍기'입니다.
* **함수(Function)**: 재료를 넣으면 맛있는 결과물이 나오는 '자동 요리 기계'입니다.

### 2. 클래스와 객체 (틀과 물건)
* **클래스(Class)**: 똑같은 물건을 여러 개 만들기 위한 '설계도' 혹은 '틀'입니다.
* **\_\_init\_\_ 및 self**: 로봇이 처음 만들어질 때 "안녕! 내 이름은 로봇1이야"라고 자기소개를 준비하는 과정입니다.
* **속성과 메서드**: 로봇의 색깔(속성)과 로봇이 발사하는 레이저(메서드)를 미리 정해두는 것입니다.

### 3. 파일 입출력 및 예외 처리 (기록과 안전제일)
* **파일 입출력**: 오늘 있었던 점수를 일기장(파일)에 적어두고, 내일 다시 읽어보는 과정입니다.
* **JSON**: 친구와 일기를 바꿔 볼 때, 서로 알아볼 수 있게 약속한 '비밀 글자 모양'입니다.
* **예외 처리(try/except)**: 실수로 엉뚱한 버튼을 눌러도 컴퓨터가 "깜짝이야!" 하고 놀라지 않게 미리 준비하는 '안전 그물'입니다.

### 4. Git 기초 (타임머신 타기)
* **Git의 필요성**: 실수로 코드를 지워도 다시 과거로 돌아갈 수 있게 도와주는 든든한 친구입니다.
* **주요 명령어**:
    * `init`: 이제부터 사진을 찍을 준비를 합니다.
    * `add`: 사진 찍을 물건을 책상 위에 올립니다. (장바구니)
    * `commit`: "찰칵!" 사진을 찍어 기록을 남깁니다.
    * `push`: 찍은 사진을 친구들도 볼 수 있게 인터넷 앨범(GitHub)에 올립니다.
    * `pull`: 친구가 올린 새로운 사진을 내 앨범으로 가져옵니다.
    * `clone`: 친구의 앨범을 통째로 복사해 옵니다.
* **브랜치와 병합**: 원래 길에서 잠시 옆길로 가서 새로운 놀이터를 만들고(Branch), 다 만들면 원래 길과 연결하는(Merge) 마법입니다.
